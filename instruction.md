# Улучшения для Animated Lab Site

## Промпт для ChatGPT / Claude

```
Ты — эксперт по frontend и UI/UX анимациям. Улучши этот сайт лаборатории.
Вот HTML-код: [ВСТАВЬ КОД]
```

---

## Список улучшений

### 1. Параллакс на slide-0
Добавь mousemove-параллакс на blur-шары в фоне — они должны двигаться в сторону,
противоположную курсору, на 20–30px.

### 2. Анимированный счётчик статистики
На slide-1 под карточками добавь 3 цифры (например: «1,247 исследований»,
«98.4% точность», «12 лет опыта»), которые анимированно считаются от 0 до значения
при появлении в viewport.

### 3. Живое время в футере slide-1
Замени статичный `UTC: 12:44:09` на реальное время, обновляющееся каждую секунду
через `setInterval`.

### 4. Stagger-анимация строк таблицы на slide-3
Каждая строка таблицы появляется с задержкой 80ms друг за другом через
`IntersectionObserver`, с `translateX(-20px) → translateX(0)` + `opacity 0 → 1`.

### 5. Hover-эффект на карточках slide-1
При наведении карточка плавно приподнимается (`translateY(-8px)`) и появляется
свечение снизу:

```css
box-shadow: 0 20px 40px rgba(19, 182, 236, 0.2);
```

### 6. Typing-эффект на заголовке slide-4
Текст «Записаться на исследование» печатается посимвольно при появлении секции
в viewport (скорость: 60ms/символ).

### 7. Валидация формы на slide-4
- При нажатии «Book Now» с пустым или невалидным email — кнопка трясётся
  (`keyframe`-анимация shake) и поле подсвечивается красным `border`.
- При валидном email — кнопка становится зелёной с галочкой на 2 секунды.

### 8. Фикс активного пункта Dynamic Island
Добавь логику: при скролле в viewport нужного `section` соответствующая ссылка
в `.island-hidden` подсвечивается цветом `primary` (`#13b6ec`).
